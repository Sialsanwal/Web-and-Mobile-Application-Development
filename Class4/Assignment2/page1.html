Empty Element

Empty Elements are elements that do not have element content and end tag.

The <Br> tag defined a line break and is an empty element without the closing tag.

Code:

<p> This is a <br /> paragraph with a line break </p>

Output:

This is a

Paragraph with a line break.

A list of commonly used Empty Element

  1. <meta />
  2. <link />
  3. <img />
  4. <br />
  5. <hr />
  6. <input />

The best practise in HTML Empty element is to always put a forward slash / sign before the greater than > sign.

In this way they are closed at their starts tags. These tags are called self-closing tags.

HTML is Not Case Senstive

HTML tags are not case sensitive: <P> means the same as <p>.<H1> means same as the <h1>

The HTML standard does not require lowercase tags but W3C recommands lower case in HTMK and demands lowercase for stricter document type like XHTML.

We will always use lowercase tag names.
