Attributes

The HTML attributes are used to provide more information to an HTML element.

  • HTML attributes only appears at start tags.It will never be on end tags.
  • All HTML elements can have attributes.
  • HTML elemets can have multiple attributes
  • HTML attributes usually come in name/value pair like: name="value"
    • An Attributes name
    • An equal = sign
    • A value surrounded by quotation marks "value"

      • you can also use single quotation marks depending on the sitution esp.When the value contains double quote.

There are two ways to specify the URL in the href attributes

  1. Abbsolute URL: Links to an external website that is hosted on another server.
  2. Relative URL: Links to an HTMLpages that is hosted within the website.
    Here, the URL does not include the domain name. If the URL begins without a slash, it will be the relative page Example:
    href="login.html"
