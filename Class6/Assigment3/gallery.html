HTML:5 Figure and Figure Caption

<figure></figure>

images often come with captions. HTML5 has introduced a new element to contains images and their caption so that the two are associated.
you can have more than one image inside the element as long as they all share the same caption.

<figcaption></figcaption>

The element has been added to HTML5 in order to allow web page authors to add a caption to an image.
Before these element were created there was no way to associated an element with its caption.
Older browser that do not understand HTML5 elements simply ignore the new elements and display the content of them.
