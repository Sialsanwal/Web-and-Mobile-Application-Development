Introduction

This is a muli-part assignment with objective of making you comfortable with HTML and CSS. By the end of this assignment, you will have styled some rectangular blocks and created a simple version of Twitter's home page.

If this assignment seems overwhelming to you, please come see a TA at TA hours to talk through some strategies for tracking it. We expext this assignment to be on time-consuming assignment as we cover lot of fundamental techniques. But with a good strategies, it can be finsihed in a reasonable amount of time.

Note: Only CSS and HTML will be used for this assignment. If you want to used JavaScript(or libraries such as jQuery) then feel free to , bu we only be grading correctness on your CSS and HTML.
